Section

Usado para divisões gerais de texto?

De acordo com a w3schools, section é usado para agrupar o conteúdo de um mesmo assunto, normalmente acompanhado de um heading <h1>

Ela é usada em:

  • Capítulos
  • Introduções
  • Novos itens
  • Informações para Contato

Article

Não sei a diferença disso para section

São para conteúdos também, só que são conteúdos independentes..? Tipo, o section é só para dividir 1 assunto, ou seja, essas sections podem remeter o mesmo assunto OU precisar de umas as outras para fazerem sentido.

Já o article, se vc pegar o conteúdo de 1 article você verá que por si só não precisa de outros conteúdos - ele mesmo já tem um significado completo que independe de outros textos

Article é usado em:

  • Postagens de:
    • Fóruns
    • Blogs
  • Comentários de usuários
  • Cards de Produtos
  • Artigos de notícias/jornais

Aside

Será que Aside realmente fica na lateral automaticamente? (OBS: Não fica kkkkk)

Aprendi a usar o Emmet :D

  • nomeDaTag + Enter - Automaticamente deixa a tag feita, com ela fechada "< >" e se for a tag a ela já cria um espaço para o href
  • Tag1>Tag2 - Deixa a Tag2 já dentro da Tag1
  • Tag*3 - cria 3 Tags prontas
  • Tag{texto $} - cria o texto dentro da tag, e o "$" é numeração - se fossem feitas 4 tags, a numeração seguiria, indo de 1 a 4 de acordo com a ordem da tag criada

Outros atalhos úteis:

  • Alt Z - Para quebrar linhas longas
  • Shift Alt F - Para organizar estruturalmente o código todo (o ruim que ele espaça umas coisas meio desnecessárias, mas para projetos longos talvez seja útil)
